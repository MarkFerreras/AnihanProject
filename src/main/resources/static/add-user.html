New Account

Create a new user account for the system.

Set the required credentials and role. Personal details can be filled in now or edited later by the admin or the user themselves.

Creation Notes

Only a username, password, and role are required. Everything else has a safe default.

  • Username must not contain spaces.
  • Password requires a minimum of 8 characters.
  • Birthdate is required — age is calculated automatically.
  • Personal details can be updated after the account is created.

Account Defaults
