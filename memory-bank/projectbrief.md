# Project Brief — Anihan SRMS

## Overview
**Anihan SRMS** (Student Record Management System) is a web application for managing student records.

## Stack
- **Front-End**: HTML5, CSS3, JavaScript ES2024, Bootstrap 5.3, DataTables 2
- **Back-End**: Java 25, Spring Boot 4.0, Spring Security 7, Spring Validation, Spring Data JPA
- **Database**: MySQL 8 (Docker)
- **Build**: Gradle (Kotlin DSL)

## Current Status
Database schema development and initial UI scaffolding (Login page).
