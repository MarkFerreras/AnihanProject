# System Patterns — Anihan SRMS

## Architecture
- Spring Boot MVC (Controller → Service → Repository)
- Packages: `controller`, `model`, `repository`, `service`

## Front-End Patterns
- Bootstrap 5.3 components and utilities for layout/styling
- Custom CSS files per page (e.g., `login.css`)
- DataTables 2 for interactive data tables
- jQuery 4.0 for DOM manipulation

## Naming Conventions
- CSS classes: kebab-case with page prefix (e.g., `login-card`, `login-form`)
- HTML IDs: camelCase (e.g., `loginForm`, `username`)
- CSS custom properties for color palette
